Asjsstring Include Text In Javascript String

Asjsstring to Include Text In Javascript

BUILD_INCLUDE

 
var getStyleCss = function () {
  var css = '// BUILD_INCLUDE("./scratch/css/style.min.css")[asjsstring,breakString?width=80]';
  return css;
};

Include minified css file

Include file as javascript string.
In this case we have a file that has been minified in the build process an outputed to ./scratch/css/style.min.css. The output file style.min.css is one long line. BUILD_INCLUDE can be used to inlcude the file and make it a little more readable.

Options

[asjsstring,breakString?width=80]

AsJsString

asjsstring determines that input is to be escaped for useage as a JavaScript string.

Breakstring

The various options of breakstring allow for the breaking of lines in different ways.
If the width option of breakstring is not set then width defaults to 80.

Breakstring Width

Width=# where as # is a number, determines that each line will be broken as close to # characers as possible depending other breakstring options.

Option

breakString?width=80

Width=80 determines that each line will be broken at 80 characers.

Config

GruntFile.js

 
module.exports = function (grunt) {
  grunt.initConfig({
    build_include: {
      default: {
        src: './lib/main.js',
        dest: './scratch/main.js'
      }
    }
  });
  grunt.loadNpmTasks('grunt-build-include');
  grunt.registerTask('default', ['build_include:default']);
};
