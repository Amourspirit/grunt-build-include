Code

Text Code

Determinses the type of encoding \ decoding to apply.
Code is require to be set with with codeKind.

Text Code Base64

code: 'base64' Base64 encoding \ decoding is applied.

Text Code Base64Uri

code: 'base64uri' Base64URI encoding \ decoding is applied.

Text Code jsString

code: 'jsString' JavaScript string encoding \ decoding is applied.

Text Code None

code: 'none' - Default: No encoding \ decoding is applied.

Text Code tsString

code: 'tsString' TypeScript string encoding \ decoding is applied.

Text Code URI

code: 'uri' URI encoding \ decoding is applied.

Text Code URIComponent

code: 'uriComponent' URIComponent encoding \ decoding is applied.
