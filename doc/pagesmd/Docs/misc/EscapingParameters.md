Some parameters that can accept text as there value such `text?before=` and `text?after=` require escaps for literal values of the following.

Escapes are as follows:  

| character   |  escaped value  |
|:-----------:|:---------------:|
|     [       |       \\[        |
|     ]       |       \\]        |
|     &       |       \\&        |
|     ,       |       \\,        |
|     =       |       \\=        |
|     \\      |       \\\\       |
|     \\n     |       \\\\n      |
|    \\r\\n   |      \\\\r\\\\n  |
|     \\t     |       \\\\t      |
