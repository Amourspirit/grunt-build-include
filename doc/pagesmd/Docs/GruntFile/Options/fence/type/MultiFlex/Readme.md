Combines [Escape](../Escape/), [Tilde](../Tilde) and [Strict](../Strict/)
into a single fence.

[Escape](../Escape/) is matched first.  
[Tilde](../Tilde) is matched second.  
[Strict](../Strict/) is the final match.
